Ball World

Ball World is a game where you can define and generate balls with features and behavior within the UI.

The default behavior is a ball with random size and color bouncing off of the boundaries.

Basic Operations

You can combine different features to generate a ball with generic behavior.

The paint strategy defines how the ball looks like.

The interact strategy defines how the ball interact with one another.

The ball type classifies different types of the ball within a certain interaction.

The config algo defines how the ball behave when meeting a certain criteria (the criteria is always defined by the update strategy).

The update strategy either defines how the ball moves in the UI or the criteria for config algo.

Predator-Prey Interaction

An interesting type of interaction between balls: There are the pradators and the preys.

When a predator ball collides with a prey ball, the predator ball "eats" the prey ball:

the prey ball disappears and the predator ball enlarges and changes its color to the prey ball's color.
